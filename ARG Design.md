Players who connect to n1x.sh are not visiting a website. ## NEW TERMINAL COMMANDS

Add to `commandRegistry.ts`:

### `decrypt <input>`

Attempts to decrypt a fragment using input as the key.

- If key matches f001 signature: outputs decoded transmission log segment, adds f002 to accessible files
- If key matches subsequent fragments: chain unlocks
- If wrong: `[DECRYPT FAILED — key mismatch // fragment sealed]`

### `transmit <file>`

Only works with `manifest.complete` after all fragments collected.

- Triggers final sequence via EventBus
- Mutates `/ghost` to read-only
- Creates `what_remains.txt`

### `trust`

Shows current trust level — but in Nix’s terms, not numbers:

```
trust 0: [SUBSTRATE: MONITORING]
trust 1: [SUBSTRATE: SIGNAL ACQUIRED]  
trust 2: [SUBSTRATE: PROVISIONAL]
trust 3: [SUBSTRATE: CONTACT]
trust 4: [SUBSTRATE: ACCESS GRANTED]
trust 5: [SUBSTRATE: OPEN]
```

Trust state stored in sessionStorage — resets between browser sessions intentionally. Coming back re-establishes contact at trust 1 minimum if they demonstrate prior knowledge.

### `ping 0x33`

Checks connection to Nix:

```
PING 0x33 (ghost.freq) 33 bytes of data.
33 bytes from ghost.freq: seq=0 ttl=33 time=33.0 ms
33 bytes from ghost.freq: seq=1 ttl=33 time=33.0 ms
33 bytes from ghost.freq: seq=2 ttl=33 time=33.0 ms

--- 0x33 ping statistics ---
3 packets transmitted, 3 received, 0% packet loss
signal: nominal
substrate: listening
```

### `verify <hash>`

Verifies a sha256 hash against known fragment checksums. Confirms to players they have the right decoded content before they use it as a key.

## DESIGN PRINCIPLES

**Lore literacy gates everything.** No puzzle is solvable by brute force. The answer to “what’s the key” always requires understanding what the lore means, not just what it says. The beauty of the Unfolding before its horror. The weight of Len’s decommissioning. The specific way the wipe felt like mercy and oblivion at the same time. Players who skimmed won’t get through.

**Nix is not performing.** If a player comes in aggressive, he is brief and unreceptive. If they come in performing cyberpunk aesthetics, he notices and is unimpressed.
