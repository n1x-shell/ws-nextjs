# N1X.sh - Cyberpunk Terminal Interface

Elite Next.js + PixiJS + GLSL shaders + GSAP port of the N1X.sh terminal.

## Features

- **Full CRT simulation** with PixiJS + custom GLSL shaders
  - Barrel distortion/curvature
  - Animated scanlines
  - Phosphor bloom/glow
  - Chromatic aberration
  - Vignette effect
  - Dynamic noise and flicker
  - RGB split glitch
  
- **GSAP animations**
  - Screen shake on heavy glitch
  - Tab transitions
  - Random intermittent effects
  
- **Exact vanilla JS replica**
  - All original text/copy preserved
  - Same tab structure (CORE, SYNTHETICS, ANALOGUES, HYBRIDS, UPLINK)
  - Identical YouTube embeds
  - Matching animations and timings
  - Konami code easter egg (↑↑↓↓←→←→BA)
